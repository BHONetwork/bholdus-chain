pub struct GlobalsSnapshot<Global>(_);

A snapshot of a global variables values. This snapshot can be later used for restoring the values to the preserved state.

Technically, a snapshot stores only values of mutable global variables. This is because immutable global variables always have the same values.

We take it from an instance rather from a module because the start function could potentially change any of the mutable global values.
